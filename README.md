#  JavaChat
자바로 만든 채팅 프로그램     

<br>

## 설명
* 채팅을 기반으로 사용자 간의 자유로운 카뮤니케이션을 제공하는 웹 애플리케이션
  

<br>

## 개발 기간
* 2019.10.01 ~ 2019.11.30

<br>
